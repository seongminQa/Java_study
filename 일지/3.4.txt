# 참조 타입

## 데이터 타입 분류
- 기본 타입(8가지)
- 참조 타입(배열 타입, 열거 타입, 클래스, 인터페이스)

참조 타입으로 선언된 변수는 '객체'가 생성된 메모리 번지(주소값)를 저장.

* 객체란?
= 데이터(필드) + 메소드
--> 메모리(heap)를 가상세계라고 가정했을 때, 사람이든 자동차 혹은 건물 등 각각의 객체라고 생각하고 만들 수 있다. 이 각각의 객체를 찾고 사용하려면 메모리의 위치(번지)를 알아야 한다. 이 위치를 담고 있는 것이 메모리(stack)에 있는 변수이다.

변수들은 모두 스택(stack)이라는 메모리 영역에 생성.
참조 타입 변수는 힙(heap) 메모리 영역의 객체 번지를 저장하고, 이 번지를 통해 객체를 참조한다.

## 메모리 사용 영역
메소드 영역 - 바이트 코드가 저장되는 영역
스택 영역 - 변수가 생성되는 영역
힙 영역 - 객체가 생성되는 영역
