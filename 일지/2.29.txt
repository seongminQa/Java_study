# 다른 JDK 사용법?
JavaProgramming(프로젝트) --> 우클릭 --> Properties --> Java Build Path --> Libraries --> Modulepath에 원하는 JDK로 변경해야하는데, Java 8은 Classpath에서 추가해주어야 하고, 이후 11~21은 Modulepath에서 진행한다.
--> add Library --> JRE System Library --> 원하는 JDK버전 선택후 적용.

이런식으로 버전별로 코드가 지원이 되는지 안되는지 확인이 가능하다.
